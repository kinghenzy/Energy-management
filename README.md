# Greene Brook Smart Energy Workspace Dashboard ☀️🔋

An interactive, browser-native simulation and diagnostic workspace for sizing home solar systems, modeling inductive starting surges, tracking multi-module battery runtimes, and verifying generator compatibility.

This production-ready software tool converts complex thermodynamic, chemical, and electrical physics into an intuitive, visual dashboard—giving you absolute mathematical certainty before spending a single dollar on alternative energy hardware.

---

## 🛠️ Integrated Engineering Core Modules

### 1. Dynamic Load Sizing & Runtime Simulator
Input your exact household appliance profiles to map real-time power consumption curves. Compute true **Depth of Discharge (DoD)** and calculate battery bank autonomy hours past sunset using standard chemical transfer formulas, moving far beyond deceptive marketing datasheets.

### 2. Surge & Inductive Load Analyzer
Avoid unexpected inverter hard-trips. This module tracks the intense **Locked Rotor Amps (LRA)** spikes produced by heavy inductive motors—such as deep-well borehole pumps, split-unit air conditioners, and refrigeration compressors—ensuring your system features adequate structural headroom.

### 3. Cross-Sectional Cable Sizing Array
Input operating DC bus voltages ($12\text{V}$ vs. $24\text{V}$ vs. $48\text{V}$) alongside loop distances to compute precise cross-sectional wire resistance ($I^2R$). Prevent hidden power losses and catastrophic capacitor starvation faults by identifying exactly when your wiring is dangerously undersized.

### 4. Generator Wave-Form & THD Profiler
Stop hybrid inverter grid-rejection loops. Model the erratic frequency drops and **Total Harmonic Distortion (THD)** profiles generated by mechanical speed governors on backup generator units, identifying optimal input configuration parameters before wiring connections.

### 5. Battery Authenticity Matrix
Your primary shield against gray-market hardware fraud. Input incoming component masses to evaluate structural density metrics. The application exposes down-tiered, sand-ballasted, or salvaged EV cells rewrapped as pristine Grade-A stationary components.
